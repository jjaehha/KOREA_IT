    함수 들을 만들었다
    --> 프론트 쪽에서 사용

mySQL 테이블에서 원하는 테이터 조회하기
    SELECT

행을 골라내고 싶을 때는 WHERE 절을 사용한다

SELECT 컬럼, 컬럼, ...
FROM 테이블명
[WHERE 조건]
[ORDER BY 컬럼 [ASC or DESC]];

조건 자리: 관계연산자, 논리연산자, SQL연산자

관계연산자

    >
    <
    >=
    <=
    =
    <>
        != 으로 사용가능

논리연산자

    조건1 AND 조건2: 모두 만족하는 조건
    조건1 OR 조건2: 둘중 하나라도 만족하는 조건
    NOT 조건: 조건을 만족하지 않는 조건

SQL 연산자
    BETWEEN a AND b
        a 이상 그리고 b 이하
    IN( 값1, 값2, 값3 )
        컬럼속에 있는 값이 괄호 안에 있는 값을 만족하는 조건
    LIKE 패턴
        컬럼속에 있는 값이 패턴을 만족하는조건
        패턴 자리에는 문자열로 패턴을 정할 수 있고,
            % : ~
                ex) %ev% (가운데 ev 라는 글자가 있는 컬럼)
            _ : 한자리
                ex) %ev_ (ev 뒤에 한글자가 있는 컬럼)
