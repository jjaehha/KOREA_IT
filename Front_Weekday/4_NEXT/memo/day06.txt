1. 반복되는 컴포넌트를 쉽게 렌더링 하는 방법
    배열 속 map 함수를 활용한다
    map 함수는 인자로 콜백함수 한 개가 필요하다
    map 함수의 인자로 전달되는 콜백함수는 세 개의 매개변수가 있을 수 있다,
    각각 세개의 매개변수는 순서대로
    배열의 요ㅕ소, 인덱스가 대입된다

3. api를 통해 서버로부터 데이터 받아오기
    fetch 함수 아용 ok
        나온지 얼마 되지 않았기 때문에 IE 에서는 사용할 수가 없다
        (IE는 공식적으로 종료되었기 때문에 더 이상 생각 하지 않아도 무관)
        하지만 모든 브라우저에서 사용할 수 있도록 만들길 원하는
        대기업, 중견기업 입장에서는 모든 브라우저 호한이 가능한 axios 라이브러리를 사용한다

    axios 라이브러리 설치
        npm install axios
        yarnd install axios

    axios를 통한 get 요청
        get 함수 사용
            axios.get(요청주소)
            비동기 함수 이기 때문에 요청하고, 기다리는 동안 다른 함수를 먼저 실행한다

    useEffect 훅함수
        특정 상황에서만 특정 코드를 실행시키고 싶을 때 사용하는 함수
        특정 상황이란? 최초 렌더링 시에만, 특정 컴포넌트가 리랜더링 될때만,
                      특정컴포넌트가 사라질때만
        useEffect(특정상황에서 실행시키고자 하는 함수, 의존성배열)

        ex)
            useEffect(() => {console.log('안녕')})
