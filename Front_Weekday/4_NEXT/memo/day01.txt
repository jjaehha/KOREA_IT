nodejs 설치
    React 프로그램을 만들기 위해서 반드시 nodejs를 설치해야하는 것은 아니다
    하지만 React 프로그램을 만들 때 사용할 다양한 라이브러리
    (다른 사람, 기업 이 만들어놓은 코드들)를 쉽게 설치하기 위해서는 npm 혹은 yarn 이라는
    도구가 필요하다.
    nodejs 설치하면 npm 혹은 yarn 이라는 도구를 사용할 수 있다

    LTS버전(추천) > 설치
    구글 > node js 검색 > 가장위의 링크>

    삭제
        재어판 > node js 제거
React
    React는 Facebook에서 만든 자바스크립트 기반 프론트엔드 라이브러리

next js
    자바스크립트 기반 프론트엔드 프레임워크
    (React의 업그레이드 버전)
    경로(라우팅)와 관련된 다양한 도구를 사용할 수 있다

터미널
    vs code 상단 터미널 클릭 > 새터미널

    기존 터미널을 종료 시키고 싶으면 X버튼이 아닌 쓰레기통 버튼 클릭

next 프로젝트 폴더만들기
    터미널에서 내가 프로젝트 폴더를 만들고 싶은 폴더로 이동
        cd 폴더이름/폴더이름

    npx create-next-app 프로젝트폴더이름
        *프로젝트 폴더이름은 반드시 소문자로만 띄어쓰기 없이 입력해 주세요

npm
    node package manager
    (일종의 파일 관리자)
    다른 개발자들이 코드를 작성하고, npm에 올려놓는다
    우리가 사용하고 싶은 코드(라이브러리)가 있다면 npm에서 다운로드 받을 수 있다!
    (노드js 설치하면 자동으로 설치가 된다)

node_modules 폴더
    해당 프로젝트에서 사용하는 여러가지 라이브러리 들이 들어있다

public 폴더
    여러가지 이미지들 혹은 favicon이  넣어놓은 파일

src 폴더
    소스파일 들이 들어있는 폴더

    pages 폴더
        실질적인 페이지 역할을 하는 폴더
    styles 폴더
        css 파일들이 들어있는 폴더
    components 폴더...
    추가적으로 폴더를 만들어도 된다

.gitignore 파일
    git에 업로드하지 않을 파일들에 대한 정보를 저장하는 파일
    node_modules 폴더도 깃허브에 올라가지 않는다

package.json 파일
    우리 프로젝트에 대한 설정
    
    dependenies
        프로젝트에서 사용중인 라이브러리와 버전 정보가 들어있다

보통 git에 올라간 프로젝트 폴더에는
node_modules가 없는 상태로 올라간다
이 폴더를 다운로드 받으면 필요한 라이브러리가 없는 상태기 때문에
실행을 못시킨다
하지만 어차피 프로젝트에서 필요한 라이브러리 이름과 버전 정보가
package.json 안에 들어있기 때문에
그 라이브러리 들을 설치하여 사용한다

npm에서 필요한 라이브러리 설치하기

    npm install 라이브리이름 라이브러리이름 라이브러리이름
        이때 install 명령어 대신 i 라고 써도 된다
        npm i 라이브러리이름 라이브러리이름 라이브러리이름

package.json 에 dependencies에 있는
라이브러리를 한번에 설치하는 방법
    npm install
        혹은
    npm i

npm 명령어를 사용할 때는 항상 경로를 잘 봐야 한다
반드시 본인의 프로젝트폴더로 이동하여 실행시키는 습관을 길러야 한다

next 프로젝트 실행시키기
    npm run dev
    기본적으로 localhost:3000 에서 확인할 수 있다

    가장 첫 화면으로 그려지는 내용은 pages 폴더의
    index.js 가 보여진다

index.js 는 엄연히 자바스크립트 파일,
index,js 안쪽에 쓰여진 <h1></h1> 코드는 html 코드가 아니다!
자바스크립트 코드, (Jsx 코드 이다)
