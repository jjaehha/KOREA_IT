변수의 선언
    var를 통한 선언
        var 변수명;

        재선언 가능 -->(문제점), 재할당 가능
        더이상 var를 사용하지 않는다

    let을 통한 선언
        let 변수명;

        재선언 불가능, 재할당 가능

    const를 통한 선언
        const 변수명;

        재선언 불가능, 재할당 불가능

변수 종류 정하기
    값을 나중에 다시 써야할 경우 --> 변수 사용
        해당 변수에 다른 값이 대입될 수 있어야 한다 --> let
        해당 변수에 최초로 대입된 값 외에는 다른값이 대입되면 안된다 --> const

const는 읽기 전용(read-only)으로 사용한다
    변수 속에 저장된 값을 사용만 할 수 있다

변수명 작성시 주의사항
    1. 숫자로 시작할 수는 없다
        let num2; // 가능
        let 1num; // 불가능
    
    2. 특수문자 사용 불가능
        let my-name; // 불가능
        단!
            let my_name; // 가능

    3. 띄어쓰기 불가능
        let my name; // 불가능
        let myname; // 가능

    4. 키워드(예약어)는 사용 불가능
        let var; // 불가능

권장사항
    1. 의미 있는 단어를 사용하자
        ex)
            나이: age
            이름: name
    
    2. 한글을 사용하지 않는다
        let 변수;

    3. 소문자로 시작하자
        let Day; // 객체와 비교하기 위해서...

가독성을 높이는 방법
    1. 스네이크 표기법
        let my_name;
    2. 카멜 표기법
        let myName; // 권장

promt 함수
    1. (소괄호 안의 값)인자값의 갯수, 타입
        2개(생략도 가능), 타입은 any
        맨 앞의 인자는 부연 설명자리애 출력
        두번째 인자는 입력창에 기본값

    2. 함수의 동작
        promt 창을 띄어주고, 사용자가 입력한 값을 받아온다

    3. 함수 사용 결과(return / 변환) 값의 타입

document.write 함수
    1. 함수인자
        몇개가 오던 상관 없으며, String 타입

    2. 함수 동작
        HTML 문서 안쪽에다 인자로 받은 값을 출력한다

    3. 함수사용결과값
        undefined

alert()
    1. 1개 any
    2. alert 창을 띄워주고 인자로 받은 값을 보여준다
    3. undefined

confirm
    1. 1개 String
    2. confirm 창을 띄어주고 확인 혹은 취소 버튼을 클릭한다
    3. 확인이면 trur, 취소면 false (Bolean 타입)

`` 기호
    `` 기호 안에 ${} 를 쓰면 {}안쪽은 자바스크립트 명령어로 동작하는 곳이고,
    변수라면 변수속의 값, 함수라면 함수 사용 결과 값 을 `` 안의 하나의 문자열로 만들어 준다

특수문자를 포함한 문자열 만들기
    '배상엽은 생가갰습니다 '배고프다' 라고';
    "배상엽은 말했습니다 "배고프다" 라고";
    '나는
    오늘이 행복하다'

    문자열 내에서 특수문자를 표현하기 위해서는 제어문자를 활용한다

제어문자
    \': 문자열 값 ' 를 의미(문자열을 알려주는 역할에서 벗어난다)
    \": 문자열 값 " 를 의미
    \n: 엔터를 의미
    \t: Tab 키를 의미
    \\: \를 의미
