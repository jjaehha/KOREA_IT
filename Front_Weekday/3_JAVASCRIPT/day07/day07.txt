반복문
    for문
    while문
    do~while문 (잘 안쓰임)

    for문
        특정 횟수만큼 반복을 시키고 싶을 때 사용

    while문
        특정 조건동안 반복을 시키고 싶을 때 사용
        ex) 사용자가 10을 입력할 때까지 반복해줘(10이 입력되지 않았다면 반복!)

    if(조건식){
        조건식이 참일때 실행할 문장;
    }

    while(조건식){
        (조건식이 참일때 실행할 문장)반복할 문장;
    }
    while문을 만나면 조건식을 확인하고
    조건식이 참이면 안쪽으로 들어와서 문장을 실행한 뒤 다시 조건식을 확인한다
    조건식이 거짓이면 안쪽으로 들어가지 않는다

do~while 문
    do{
        반복할 문장;
    }while(조건식);

    do 영역은 최초 한 번은 조건식과 관계 없이 실행이 되고
    그 뒤에 조건식을 확인하여 반복여부를 결정한다

    ex) 문자열을 모두 대문자로 바꿔주는 기능
    'apPle' --> 'APPLE'
    100줄의 코드로 구현을 했다

    100줄짜리 코드를 미리 묶음으로 정의해놓고 함수 한 줄을 사용해야겠다!
    100줄짜리 —> myFunction
    앞으로 대문자로 바꾸고 싶을때마다 myFuction()한줄만 실행하면 된다..

    1. 인자가 몇개, 무슨 타입이 필요할까?
    2. 함수 동작은 어떻게 구현할까?
    3. 함수 사용 결과는 무슨 값으로 되돌릴(반환할, return 할)까?

함수 정의
    function 함수이름(){
        코드;
        코드;
        코드;
    }

    함수 사용(호출)
        함수호출연산자 () 를 사용한다
        log 함수 호출하기 위해서 log()
        ceil 함수 호출하기 위해서 ceil()

    매개변수
        함수를 정으이할 때 소괄호 안쪽에 선언된 변수, 이변수에는
        함수를 호출 할 때 넘겨준 인자 값이 대입된다.

        사용이유: 함수를 사용하는 사람ㅇ이 값을 전달해줘야 할 때 사용한다
        만일 값을 넘겨주지 않으면 매개변수에는 undefined 값이 대입된다

        이때 매개변수를 만들 대 대입연산자를 사용하여 기본 값을 설정 할 수 있다
        (보통 기본매개변수는 뒤쪽부터 차례로 작성하여 optional 하다는 것을 알려주는 편이다)

    가변 매개변수
        console.log 함수의 인지는 몇개? 가변(갯수가 가변적으로 변한다)


    함수의 return
        함수 사용 결과를 값으로 되돌려주는 명령어
        return 이 실행되면 함수는 return 옆에 있는 값으로 반환되면
        그 즉시 함수가 종료된다.

        return은 우선순위가 낮기 떄문에 가장 마지막에 해석한다
