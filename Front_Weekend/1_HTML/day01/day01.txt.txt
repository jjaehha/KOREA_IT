-UI, UX
	UI (USER INTERFACE)
		사용자가 보는 레이아웃(화면)
		ex) 이미지, 폰트, 색상, 글, 그림, 버튼
	
	UX (USER EXPERIENCE)
		사용자가 어플리케이션을 이용함으로서 느낄 수 있는 만족감
		ex) 회원가입을 하는 동안 상용자가 불편함이 없었는가

		즉, UI, UX를 고려한다는 것은 사용자가 보고 있는화면이
		사용자가 상용하였을때 불편함이나 제약 사항이 없는지
		고려하여 만든다는 것

HTTP (Hyper Text Transfer Protocol)

인터넷에서 데이터를 서로 주고 받을 수 있도록
사용하는 일종의 규칙, 규약

따라서 도메인 앞에 http, https가 붙으면
나는 이규약을 사용하여 request와 response를 주고 받겠다

http request message
http response message

s의 차이는 보안적 요소가 추가적으로 들어가있는가의 차이

플러그인 설치
1. Korean Laguage pack // 한글패치
